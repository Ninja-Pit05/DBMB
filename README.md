# DBMB 2.1
The main repository for the development of Droneboi Map Bot.

The **Droneboi Map Bot**, also known as **DBMB**, is a discord tool for the mobile game series [Droneboi: Conquest](https://play.google.com/store/apps/details?id=com.rizenplanet.droneboiconquest) by [Rizen Planet Studios](https://play.google.com/store/apps/dev?id=7611018448877408213).
DBMB's main functionality is to be a quick, intuitive and easy way to check in-game faction territory and be a useful tool to the Droneboi community on discord.
